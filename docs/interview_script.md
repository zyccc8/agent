# 3-5 分钟项目讲解稿

大家好，这个项目是一个 AI 驱动的竞品分析 Agent 协作系统。我选择竞品分析这个场景，是因为传统流程高度依赖人工搜索、整理和判断，信息源分散且结论很难追溯。

系统的核心不是单次让大模型写报告，而是把数字调研小组拆成多个 Agent：采集 Agent 负责公开信息，清洗 Agent 负责结构化证据，分析 Agent 负责竞品画像和横向比较，报告 Agent 负责输出结构化报告，质检 Agent 负责检查来源、置信度和重复结论。

我在第一版里没有追求复杂爬虫，而是优先实现完整闭环。用户输入行业和竞品后，系统会生成报告、证据索引、DAG 工作流和每个 Agent 的运行日志。这样面试官可以看到每个分析结论从哪里来，也能看到系统如何处理缺失来源。

这个项目后续可以继续扩展：接入真实搜索和网页抽取，用 LangGraph 表达更复杂的工作流，引入数据库保存多次运行结果，并加入大模型生成更深入的商业洞察。第一版重点展示我对 Agent 协作、结构化数据、可观测性和可追溯 AI 应用的理解。
